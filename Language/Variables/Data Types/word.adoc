= word

// OVERVIEW SECTION STARTS
[#overview]
--

[float]
=== Description
A word can store an unsigned number of at least 16 bits (from 0 to 65535).
[%hardbreaks]


[float]
=== Syntax
`word var = val;`


[float]
=== Parameters
`var`: variable name +
`val`: the value to assign to that variable
