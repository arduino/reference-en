[float]
=== Description
Long variables are extended size variables for number storage, and store 32 bits (4 bytes), from -2,147,483,648 to 2,147,483,647.

If doing math with integers, at least one of the numbers must be followed by an L, forcing it to be a long. See the link:../integerConstant[Integer Constants] page for details.
[%hardbreaks]

[float]
=== Syntax

`long var = val;`

`var` - the long variable name
`val` - the value assigned to the variable
[%hardbreaks]
