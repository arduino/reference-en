= length()


// OVERVIEW SECTION STARTS
[#overview]
--

[float]
=== Description
Returns the length of the String, in characters. (Note that this doesn't include a trailing null character.)

[%hardbreaks]


[float]
=== Syntax
[source,arduino]
----
string.length()
----

[float]
=== Parameters
`string`: a variable of type String


[float]
=== Returns
The length of the String in characters.
