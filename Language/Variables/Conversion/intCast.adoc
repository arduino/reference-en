= int()


// OVERVIEW SECTION STARTS
[#overview]
--

[float]
=== Description
Converts a value to the link:../../data-types/int[int] data type.
[%hardbreaks]


[float]
=== Syntax
`int(x)` +
`(int)x` (C-style type conversion)


[float]
=== Parameters
`x`: a value. Allowed data types: any type.


[float]
=== Returns
Data type: `int`.
