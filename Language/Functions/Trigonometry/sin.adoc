= sin(rad)


// OVERVIEW SECTION STARTS
[#overview]
--

[float]
=== Description
Calculates the sine of an angle (in radians). The result will be between -1 and 1.
[%hardbreaks]


[float]
=== Syntax
`sin(rad)`


[float]
=== Parameters
`rad`: The angle in radians. Allowed data types: `double`.


[float]
=== Returns
The sine of the angle. Data type: `double`.
