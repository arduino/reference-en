= cos(rad)


// OVERVIEW SECTION STARTS
[#overview]
--

[float]
=== Description
Calculates the cosine of an angle (in radians). The result will be between -1 and 1.
[%hardbreaks]


[float]
=== Syntax
`cos(rad)`


[float]
=== Parameters
`rad`: The angle in radians. Allowed data types: `double`.


[float]
=== Returns
The cos of the angle. Data type: `double`.
