= bitSet()


// OVERVIEW SECTION STARTS
[#overview]
--

[float]
=== Description
Sets (writes a 1 to) a bit of a numeric variable.
[%hardbreaks]


[float]
=== Syntax
`bitSet(x, n)`


[float]
=== Parameters
`x`: the numeric variable whose bit to set. +
`n`: which bit to set, starting at 0 for the least-significant (rightmost) bit.


[float]
=== Returns
Nothing
